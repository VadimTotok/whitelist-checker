# whitelist-checker (WLC)

**WLC** — это кроссплатформенная утилита на Go для быстрой диагностики интернет-доступа. Она помогает определить, есть ли проблемы с подключением или активны ли сетевые ограничения.

## Что такое белые списки?

Белые списки — это перечни интернет-ресурсов (веб-сайтов, приложений, научных журналов), доступ к которым должен сохраняться даже при ограничениях в сети.  
Такие списки формируются Минцифры и включают социально значимые сервисы:

- Госуслуги
- ВКонтакте
- Яндекс
- Ozon
- Банковские системы
- СМИ и др.

## Как работает WLC?

Утилита проверяет доступность двух групп сайтов:

- **Белый список** — ключевые российские ресурсы (например, Яндекс, ВК, Госуслуги).  
  Если они недоступны — вероятна проблема с интернет-соединением.
- **Обычные сайты** — популярные зарубежные сервисы (Google, YouTube, GitHub).  
  Если они недоступны, а белый список работает — возможно, активны блокировки или фильтрация трафика.

## Что проверяет WLC?

- DNS-разрешение
- TCP-соединение (порты 80 и 443)
- HTTP-ответ

На основе этих данных WLC формирует краткий отчёт о состоянии подключения.

## Установка и запуск

1\. Скачайте подходящую версию из раздела [Releases](https://github.com/VadimTotok/whitelist-checker/releases).

2\. Запустите утилиту:

### Windows
ВРЕМЕННО: запустите  wlc.exe из cmd или PowerShell, а не двойным кликом
```bash
cd C:\path\to\program
.\wlc.exe
```
### Linux
```bash
chmod +x wlc-linux
./wlc-linux
```
